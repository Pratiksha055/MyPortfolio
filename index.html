GrowingSnake- FunGame

This is a fully responsive, browser-based implementation of the classic Snake game, built using HTML5 Canvas, CSS3, and vanilla JavaScript. The game grid is dynamically calculated to adapt to different screen sizes, making it playable on both desktop and mobile devices. The logic handles directional control, collision detection, dynamic food placement, and real-time snake movement.

It features a glowing, animated snake head, sound effects triggered by game events (eating, game over, win), and high score tracking via localStorage. The game loop runs using setInterval, and includes clean separation of concerns across HTML, CSS, and JS for maintainability and scalability. A win condition is implemented when the snake fills all available grid cells, triggering a celebratory sound and message.
